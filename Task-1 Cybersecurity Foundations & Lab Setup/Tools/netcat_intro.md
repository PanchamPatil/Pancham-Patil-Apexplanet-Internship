# Netcat (nc) Introduction

Netcat is a versatile networking tool used for reading and writing data across network connections.

---

## Why Use Netcat?
- Port scanning
- Banner grabbing
- File transfer
- Network debugging

---

## Common Netcat Commands

### Port Scan
```bash
nc -zv 192.168.1.1 1-1000

Banner Grabbing
nc 192.168.1.1 80

Listening on a Port
nc -lvp 4444

Netcat in Cybersecurity

Netcat is often called the "Swiss Army Knife" of networking due to its versatility.
