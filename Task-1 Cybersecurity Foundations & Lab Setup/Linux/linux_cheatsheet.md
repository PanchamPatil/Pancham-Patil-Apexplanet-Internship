# Linux Cheat Sheet (Cybersecurity Focus)

## File & Directory
| Command | Description |
|------|------------|
| ls | List files |
| cd | Change directory |
| pwd | Show current path |
| mkdir | Create directory |
| rm | Delete files |

---

## Permissions
| Command | Description |
|------|------------|
| chmod | Change permissions |
| chown | Change ownership |
| ls -l | View permissions |

Permission format:  
r = read, w = write, x = execute  

---

## Package Management
| Command | Description |
|------|------------|
| apt update | Update package list |
| apt install | Install package |
| apt remove | Remove package |

---

## Networking
| Command | Description |
|------|------------|
| ifconfig | Network interfaces |
| ip a | IP address |
| ping | Connectivity test |
| netstat | Network stats |
| traceroute | Trace route |

---

## System Information
| Command | Description |
|------|------------|
| whoami | Current user |
| uname -a | System info |
| df -h | Disk usage |
| free -h | Memory usage |

---

## Process Management
| Command | Description |
|------|------------|
| ps | Running processes |
| top | Real-time processes |
| kill | Stop process |

---

## Important Notes
- Always use `sudo` carefully
- Avoid running unknown scripts
- Understand commands before execution

---

## Conclusion
This cheat sheet helps in daily Linux usage and penetration testing tasks in Kali Linux.
