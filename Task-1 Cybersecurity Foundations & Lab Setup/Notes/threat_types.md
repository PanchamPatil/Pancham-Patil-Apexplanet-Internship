# Cybersecurity Threat Types

## 1. Phishing
Phishing is a social engineering attack where attackers trick users into revealing sensitive information using fake emails or websites.

## 2. Malware
Malware is malicious software designed to damage or gain unauthorized access to systems.

### Types of Malware:
- Virus
- Worm
- Trojan
- Spyware
- Adware

## 3. Ransomware
Ransomware encrypts victim data and demands payment to restore access.

## 4. DDoS (Distributed Denial of Service)
DDoS attacks flood a system with traffic, making it unavailable to legitimate users.

## 5. SQL Injection
SQL Injection occurs when attackers manipulate database queries to access or modify data.

## 6. Brute Force Attack
Attackers try multiple username and password combinations to gain access.

## 7. Insider Threat
Threats caused by trusted individuals within an organization, either intentionally or unintentionally.

---

## Summary
Understanding threat types helps organizations design better defensive strategies and improve overall security posture.
